Projeto: Padaria do Bairro
Descrição do Projeto:
Website institucional desenvolvido para a "Padaria do Bairro". O objetivo deste projeto é aplicar os conceitos fundamentais de HTML5 Semântico, estruturação de tabelas de dados, e incorporação acessível de mídias (imagens e vídeos).

Recursos de Acessibilidade e Boas Práticas (Atividades 1 a 4):
1. Estrutura Semântica: Utilização correta de tags HTML5 como <header>, <main>, <section> e <footer> para facilitar a navegação.
2. Tabelas Acessíveis: O cardápio e os horários foram construídos utilizando as tags de agrupamento <thead> e <tbody>, com cabeçalhos bem definidos (<th>).
3. Textos Alternativos (alt): Todas as tags <img> possuem o atributo "alt" com descrições ricas e detalhadas do que está acontecendo na imagem (ex: "Vista superior de uma mesa com várias pessoas...").
4. Fallback de Vídeo: A tag <video> inclui controles nativos de reprodução e um texto alternativo embutido para informar usuários que utilizam navegadores incompatíveis.
5. Nomenclatura Padronizada: Pastas e arquivos foram renomeados rigorosamente utilizando nomes descritivos, com letras minúsculas e sem acentos (separados por hífen), garantindo excelente legibilidade.
