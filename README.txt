Projeto: Padaria do Bairro

Descrição do Projeto:
Este projeto consiste na criação da página inicial (index.html) para uma padaria local, estruturada utilizando as melhores práticas do HTML5 semântico. O objetivo é apresentar o negócio, seus produtos, diferenciais e informações de contato de forma clara e acessível.

Estrutura de Pastas:
/padaria-do-bairro
  ├── /css/         (Pasta reservada para futuros arquivos de estilo - CSS)
  ├── /js/          (Pasta reservada para futuros scripts - JavaScript)
  ├── /images/      (Pasta reservada para as imagens do site)
  └── index.html    (Arquivo principal contendo a estrutura da página)

Estrutura HTML Utilizada:
- Tags Semânticas: Foram utilizadas as tags <header>, <main>, <section> e <footer> para organizar o conteúdo de forma lógica.
- Hierarquia de Títulos: O nome da padaria foi destacado com <h1>, enquanto as grandes seções receberam <h2> e subtítulos menores <h3>.
- Listas: As listas não-ordenadas (<ul>) foram usadas para produtos e diferenciais, e a lista ordenada (<ol>) foi aplicada aos horários de funcionamento.
- Comentários: O código-fonte contém comentários explicativos detalhando a função de cada bloco de código.
